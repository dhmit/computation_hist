Magnetic Core Memory

Unlike the computers that came before the Whirlwind, the Whirlwind did not use vacuum tubes to store information. Vacuum tubes were able to store bits in 0 and 1; however, the vacuum tubes were prone to breakage and wore out quite frequently. To overcome this problem, Jay Forrester, the lead scientist behind the creation and design of the Whirlwind, created the magnetic core memory system. The magnetic core memory system used a grid of magnets to store information in bits based on the direction of the polarity of the magnets. The system was successful and was utilized until the creation of modern-day RAM in the 70s.
